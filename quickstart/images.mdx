1. Go to the Personal Center to obtain your image upload credentials.

2. In the docker environment where your image is located, perform the following operations:

   1. Log in to the InfrAI Image Service Docker Registry


   2. Push the image to the Registry


   ```bash
   docker tag [imageId] image.infrai.com/dev-xgmdbroksktnnznsisri/[image]:[tag]
   ```

